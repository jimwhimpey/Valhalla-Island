--- layout: post title: Adjective Noun Pairs ---

I’ve been thinking about adjective noun pairs that are used almost exclusively together. It’s hard to find a noun that’s only used when it’s being described by one particular adjective but it’s easier to find an adjective that’s almost only used to describe one noun. Here’s what I’ve got so far:

  • Avid Photographer
  • Moot Point
  • Cold-blooded Killer
  • Robust API
  • Gaping Void
  • Canonical Example
