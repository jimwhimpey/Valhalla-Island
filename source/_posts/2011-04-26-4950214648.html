Here’s a niche thing I made to scratch my own cycling itch: Frugal Rouleur.

I love working on these projects where I get to do everything myself. For this I created a REST API to interface with the online stores, wrote JavaScript that calls the API and designed the front end using my current favourite tools: Sinatra, HAML, SASS and Heroku for hosting.
